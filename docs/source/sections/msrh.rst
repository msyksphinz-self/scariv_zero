MSRH RISC-V CPUコア外観
=======================

MSRHはRISC-V互換のアウトオブオーダ実行可能なCPUです。

マイクロアーキテクチャの詳細は以下の通りです。

- RISC-V RV32 / RV64の両方をサポート
  - RV32GC (RV32IMAFDC)をサポート
    - Memo: AFDは不完全
  - RV64GC (RV64IMAFDC)をサポート
    - Memo: AFDは不完全
- アウトオブオーダ実行可能なパイプライン

実装の特徴は以下の通りです。

- SystemVerilog記述
- パラメタライズされた柔軟なコア構成
  - コア内の多くのパラメータが変更可能で、面積・性能を調整した最適なコアを生成可能
